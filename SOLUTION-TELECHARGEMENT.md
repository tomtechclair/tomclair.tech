# 🔧 Solution Problème de Téléchargement Git

## 🚨 Problème :
Le téléchargement automatique de Git ne fonctionne pas

## ✅ Solutions alternatives :

### 🎯 Solution 1: Installation manuelle (recommandée)
1. **Double-cliquez** sur `install-git-simple.cmd`
2. **Le site Git s'ouvre** automatiquement
3. **Téléchargez** Git pour Windows
4. **Installez** avec les options par défaut
5. **IMPORTANT** : Cochez "Add Git to PATH"
6. **Redémarrez** votre ordinateur

### 🎯 Solution 2: GitHub Desktop (plus simple)
1. **Double-cliquez** sur `github-desktop-alternative.cmd`
2. **Téléchargez** GitHub Desktop
3. **Installez** GitHub Desktop
4. **Ouvrez** le dossier `tomclair.tech`
5. **Commit** → **Push** en 2 clics

### 🎯 Solution 3: Via le Microsoft Store
1. Ouvrez le Microsoft Store
2. Cherchez "Git"
3. Cliquez sur "Installer"
4. Attendez l'installation

## 📋 Instructions détaillées - Solution 1 :

### Étape 1: Lancer le script
```
Double-cliquez sur : install-git-simple.cmd
```

### Étape 2: Télécharger
1. Le site https://git-scm.com/ s'ouvre
2. Cliquez sur "Download for Windows"
3. Enregistrez le fichier

### Étape 3: Installer
1. Double-cliquez sur le fichier `.exe`
2. Cliquez "Run" si demandé
3. **Cochez "Add Git to PATH"** (très important !)
4. Cliquez "Next" sur tous les écrans
5. Cliquez "Install"
6. Cliquez "Finish"

### Étape 4: Vérifier
1. **Redémarrez** votre ordinateur
2. Ouvrez une nouvelle fenêtre CMD
3. Tapez : `git --version`

### Étape 5: Utiliser
1. Double-cliquez sur `update-github.cmd`
2. Votre site sera mis à jour automatiquement !

## 📱 Instructions détaillées - Solution 2 (GitHub Desktop) :

### Étape 1: Lancer le script
```
Double-cliquez sur : github-desktop-alternative.cmd
```

### Étape 2: Installer GitHub Desktop
1. Le site GitHub Desktop s'ouvre
2. Téléchargez et installez GitHub Desktop
3. Ouvrez GitHub Desktop

### Étape 3: Configurer le projet
1. Cliquez sur "Add an Existing Repository"
2. Naviguez vers : `C:\Users\tomg-\OneDrive\Bureau\Météo\tomclair.tech`
3. Cliquez sur "Add Repository"

### Étape 4: Mettre à jour le site
1. Les changements apparaissent dans GitHub Desktop
2. Écrivez un message de commit
3. Cliquez sur "Commit to main"
4. Cliquez sur "Push origin"
5. Votre site est mis à jour !
