# 🎉 Guide Post-Installation Git/GitHub Desktop

## ✅ Félicitations ! Installation en cours...

Pendant que Git/GitHub Desktop s'installe, voici ce qu'il faut faire ensuite :

## 🔄 Étapes immédiates après installation :

### 1. **Redémarrer votre ordinateur**
- Très important pour que Git soit reconnu
- Ou ouvrez simplement une nouvelle fenêtre CMD

### 2. **Vérifier l'installation**
Ouvrez une nouvelle CMD et tapez :
```cmd
git --version
```

### 3. **Configurer Git (une seule fois)**
```cmd
git config --global user.name "tom-visionai-pro"
git config --global user.email "votre-email@example.com"
```

## 🚀 Utilisation selon votre choix :

### Si vous avez installé **Git en ligne de commande** :
1. Double-cliquez sur `update-github.cmd`
2. Le script mettra tout à jour automatiquement
3. Votre site sera en ligne en 2-3 minutes

### Si vous avez installé **GitHub Desktop** :
1. Ouvrez GitHub Desktop
2. Cliquez sur "Add an Existing Repository"
3. Naviguez vers : `C:\Users\tomg-\OneDrive\Bureau\Météo\tomclair.tech`
4. Les changements apparaissent automatiquement
5. Cliquez sur "Commit to main" puis "Push origin"
