# Instructions pour GitHub Desktop

## Étapes pour créer le dépôt GitHub

1. **Ouvrir GitHub Desktop**
   - Lancez l'application GitHub Desktop
   - Connectez-vous avec votre compte GitHub

2. **Créer un nouveau dépôt**
   - Cliquez sur "File" > "New Repository"
   - Nom : `weather-app` ou `meteo-app`
   - Description : "Application météo moderne avec interface iOS-like"
   - Choisissez "Public" ou "Private"
   - Ne cochez pas "Initialize with README"

3. **Ajouter le dossier du projet**
   - Cliquez sur "Add Local Repository"
   - Sélectionnez le dossier : `C:\Users\tomg-\OneDrive\Bureau\Météo\your-daily-forecast-main`
   - Cochez "Publish this repository" (optionnel)

4. **Premier commit**
   - GitHub Desktop va automatiquement créer un commit initial
   - Titre du commit : "Initial commit - Weather App iOS-like"
   - Description : "Application météo moderne avec React et TypeScript"

5. **Publier sur GitHub**
   - Cliquez sur "Publish repository"
   - L'application sera disponible sur : `https://github.com/VOTRE_NOM/weather-app`
